Talkd.ai is an open-source software that enables you to easily deploy any LLM available through LangChain.

It is currently in beta, and we are working on making it as easy as possible to deploy your own LLM.

For now, we have 2 main repositories:

  • Dialog - An extensible LLM API that enables you to easily deploy any LLM.
  • WhatsApp Plugin - The first plugin we developed that enables users to launch Dialog with WhatsApp.
